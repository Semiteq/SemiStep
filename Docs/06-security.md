# 6. Требования по безопасности и разграничению прав

## 6.1. Роли пользователей

| Роль | Описание | Права |
|------|----------|-------|
| **Оператор** | Запуск и мониторинг рецептов | Только просмотр, запуск/остановка |
| **Технолог** | Разработка рецептов | Редактирование, сохранение |
| **Инженер** | Настройка системы | Полный доступ, конфигурация |
| **Администратор** | Управление пользователями | Управление ролями и правами |

---

## 6.2. Разграничение доступа

### Операции редактирования

- Создание/удаление шагов
- Изменение параметров
- Сохранение рецепта

### Операции исполнения

- Загрузка рецепта в ПЛК
- Запуск/остановка исполнения
- Аварийная остановка

### Операции администрирования

- Настройка подключения к ПЛК
- Управление пользователями
- Просмотр логов
